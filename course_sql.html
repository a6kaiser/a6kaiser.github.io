Introduction to SQL

Welcome to the Introduction to SQL course. In this course, we'll cover the basics of SQL, including how to query databases, manipulate data, and more.

Chapter 1: Introduction to Databases

In this chapter, we'll learn about what databases are and why they're important.

1.1 What is a Database?

A database is an organized collection of data, generally stored and accessed electronically from a computer system.

1.2 Relational Databases

Relational databases are based on the relational model of data, which organizes data into tables (relations) of rows and columns.

Chapter 2: Basic SQL Queries

This chapter introduces basic SQL queries for retrieving data.

2.1 SELECT Statement

The SELECT statement is used to select data from a database.

SQL 

SELECT column1, column2
FROM table_name;

2.2 WHERE Clause

The WHERE clause is used to filter records.

SQL 

SELECT column1, column2
FROM table_name
WHERE condition;

Chapter 3: Mathematical Operations in SQL

SQL can perform various mathematical operations.

3.1 Calculating Averages

We can calculate the average of a set of values using the AVG function.

LaTeX

Formula for average:

$$ \text{Average} = \frac{\sum_{i=1}^{n} x_i}{n} $$
SQL 

SELECT AVG(column_name)
FROM table_name;

3.2 Summation

The SUM function returns the total sum of a numeric column.

LaTeX

Summation formula:

$$ S = \sum_{i=1}^{n} x_i $$
SQL 

SELECT SUM(column_name)
FROM table_name;
